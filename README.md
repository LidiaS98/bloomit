**BloomIT** is a web application for tracking daily health habits. Users can log sleep, hydration, physical activity, mood, and energy levels, then track their progress with weekly and monthly charts. The application generates personalized insights and reminders, helping them build healthier habits step by step.

Built with Java + Spring Boot (backend), PostgreSQL (database), and React.
